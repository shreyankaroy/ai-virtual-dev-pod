AI Virtual Development Pod — MVP Scope

Input:
User enters a high-level software idea.

Workflow:
1. Business Analyst Agent generates structured user stories.
2. Design Agent generates architecture and API plan.
3. Developer Agent generates backend code.
4. Testing Agent generates pytest test cases.
5. Test Runner executes tests automatically.
6. If tests fail, Developer Agent attempts one fix.
7. Project Lead Agent reviews outputs and provides summary.

Interface:
Streamlit dashboard showing:
- User stories
- Design
- Generated code
- Test results
- Review summary
